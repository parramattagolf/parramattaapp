## 1. 라운딩 목록 페이지 (`/rounds`)

### 2.1 네비게이션


### 2.2 표시 규칙

- **월별 그룹화**: 라운딩 행사를 월 단위로 그룹화하여 표시
- **유튜브 임베드**: 각 월의 마지막 행사 아래에 유튜브 재생목록 임베드 표시
- **행사 카드 스포트라이트**:
  - 참가자가 있는 경우: 초록색(emerald-500) 테두리 (1px)
  - 사전예약자만 있는 경우: 파란색(blue-500) 테두리 (1px)
  - 아무도 없는 경우: 기본 흰색 테두리

### 2.3 행사 카드 정보

- **제목**: 18px, font-black
- **설명**: 4px, 최대 3줄 표시, 초과 시 말줄임표(...)
- **날짜 정보**: 요일 및 일자 표시
- **Premium 배지**: 스폰서가 있는 경우 표시

## 7. 공통 규칙

### 8.1 네비게이션 바

- **고정 표시**: 주요 페이지(/rounds, /members, /sponsors, /my)에서는 항상 표시
- **자동 숨김**: 스크롤 다운 시 자동으로 숨김 (주요 페이지 제외)
- **페이지별 인디케이터**: 각 페이지마다 고유한 색상의 점 표시
  - 라운딩: 초록색
  - 인맥: 파란색
  - 스폰서: 노란색
  - 마이: 빨간색

### 8.2 인증

- 비로그인 사용자는 로그인 페이지로 리다이렉트
- 회원 등급에 따른 접근 제한 적용

### 8.3 반응형 디자인

- 최대 너비: 500px
- 모바일 우선 디자인
- 터치 인터랙션 최적화

## 8. 조인방 상세 페이지 (`/rounds/[id]`)

### 9.1 조인 및 퇴장 규정

- **조인 신청 시 주의사항**:
  - 신청 후 당일 취소나 노쇼 발생 시 매너 점수가 크게 차감될 수 있습니다.
- **방 나가기 기능**:
  - "정말 방을 나가시겠습니까?" 컨펌창과 함께 차감 안내 표시
  - 실행 시 DB에서 매너 점수 및 포인트 즉시 차감 및 이력 기록

### 9.2 사전예약 규정 (Pre-reservation)

- **사전예약 신청 시 보상**:
  - 참여 의사 표현에 대한 감사 보너스로 **매너점수 1점** 즉시 시상
  - 기존 참여자로부터 초대를 받을 수 있는 자격 부여
- **사전예약 취소 시**:
  - 추가적인 페널티(위약금)는 없으나, 신청 시 지급된 **매너점수 1점은 다시 회수** 처리됨
