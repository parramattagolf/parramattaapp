# 📱 파라마타 골프 앱 스토어 출시 가이드

현재 만드신 **Next.js 웹 서비스**를 **애플 앱스토어(iOS)**와 **구글 플레이스토어(Android)**에 출시하기 위한 전체 로드맵입니다.

---

## 1단계: 필수 준비물 (계정 및 비용)

스토어에 앱을 올리려면 개발자 권한이 있는 계정이 반드시 필요합니다.

| 구분 | 애플 앱스토어 (Apple) | 구글 플레이스토어 (Google) |
| :--- | :--- | :--- |
| **사이트** | [Apple Developer Program](https://developer.apple.com/) | [Google Play Console](https://play.google.com/console/) |
| **비용** | **연 $99 (약 14만원 / 매년 갱신)** | **$25 (약 3만 5천원 / 평생 1회)** |
| **준비물** | D-U-N-S 번호 (사업자의 경우), 법인 카드 | 구글 계정, 해외 결제 카드 |
| **심사 기간** | 약 1~3일 (깐깐함) | 약 1~7일 (상대적으로 빠름) |

> **Tip:** 개인 명의로 하실지, 법인(사업자) 명의로 하실지 미리 결정하셔야 합니다. 법인의 경우 D-U-N-S Number 발급 등으로 시간이 조금 더 소요됩니다.

---

## 2단계: 기술적 변환 작업 (웹 → 앱)

지금 만드신 것은 "웹사이트"이므로, 이를 "앱"으로 포장하는 기술이 필요합니다.  
Next.js에서는 **[Capacitor](https://capacitorjs.com/)**라는 도구를 가장 많이 사용합니다.

### 작업 과정
1. **Capacitor 설치**: 프로젝트에 모바일 변환 엔진을 설치합니다.
2. **네이티브 프로젝트 생성**: 명령어 한 번으로 `android` 폴더와 `ios` 폴더가 생성됩니다.
3. **기능 연결**: 
   - 카메라, 푸시 알림, 위치 정보 등 앱 고유 기능을 연결합니다.
   - 현재 사용 중인 **카카오 로그인**을 웹 방식에서 **네이티브(앱) SDK 방식**으로 수정해야 할 수도 있습니다. (앱 심사 시 "네이티브 로그인 경험"을 중요하게 보기 때문입니다.)
4. **빌드 (Build)**: 웹 코드를 앱용 설치 파일(`.ipa`, `.aab`)로 변환합니다.

> **저(AI)에게 요청하시면 기본적인 Capacitor 세팅과 안드로이드/iOS 폴더 생성까지는 바로 진행해 드릴 수 있습니다.**

---

## 3단계: 디자인 자산 준비

스토어에 올릴 때 필요한 이미지들입니다. 디자이너(혹은 저)와 함께 준비해야 합니다.

1. **앱 아이콘**: 휴대폰 바탕화면에 깔릴 아이콘 (1024x1024px)
2. **스플래시 화면**: 앱 켤 때 잠깐 나오는 로딩 화면
3. **스토어 스크린샷**:
   - 필수: 아이폰(6.5인치, 5.5인치), 아이패드(12.9인치), 안드로이드 폰 스크린샷
   - 앱의 주요 기능을 설명하는 예쁜 이미지를 만들어야 합격률이 높습니다.

## 4단계: 심사 및 출시

### iOS (애플) - 주의사항
- **로그인 정책**: 소셜 로그인(카카오 등)이 있다면, 반드시 **"Apple로 로그인"** 기능도 같이 넣어야 심사를 통과시켜 줍니다. (필수 사항)
- **웹뷰 단순 포장 금지**: 단순히 웹사이트를 앱으로만 감싸면 "앱으로서의 기능이 부족하다"며 거절(Reject)당할 수 있습니다. 푸시 알림 등 앱만의 기능을 넣었다고 어필해야 합니다.

### Android (구글) - 주의사항
- 최근 정책 변경으로, 개인 개발자 계정의 경우 **20명의 테스터를 모집하여 2주간 비공개 테스트**를 진행해야만 출시가 가능합니다. (사업자 계정은 바로 출시 가능)
